Never Miss a Birthday Again! 🎉

A Telegram bot that sends birthday reminders straight to your chat – simple, smart, and sentimental!

Try the Bot Learn More

Amazing Features

📅 Daily Reminders

Automatically sends birthday reminders every morning at 9:00 AM with age calculations and contact details.

🎂 Smart Gender Detection

Intelligently detects gender from names and uses appropriate pronouns (him/her/them) in birthday messages.

💾 JSON Data Management

Simple data.json file format with name, phone, and date of birth. Falls back to sample data for testing.

🤖 Interactive Commands

Use /today, /tomorrow, /all, and /help commands to check birthdays anytime.

📱 Contact Integration

Includes phone numbers in reminders so you can quickly call or text birthday wishes to your contacts.

🎯 Precise Age Calculation

Calculates exact age considering leap years and displays "turning X years old" for upcoming birthdays.

Bot Commands 🤖

📋 /start

Welcome message with setup instructions and available commands overview.

📅 /today

Check today's birthdays with age, phone number, and personalized pronouns.

🔮 /tomorrow

Preview tomorrow's birthdays so you can prepare your wishes in advance.

📖 /all

Display complete birthday list with formatted dates and contact information.

🆘 /help

Detailed setup instructions, data format examples, and installation guide.

Technical Features 🔧

🔄 Auto-Scheduling

Smart scheduling system that calculates time until next 9 AM reminder and sets up daily intervals.

🛡️ Error Handling

Robust error handling for file operations, API calls, and graceful fallbacks to sample data.

📊 Data Format

Simple JSON structure: {"name": "John Doe", "phone": "+1-555-0123", "dob": "1990-01-15"}
